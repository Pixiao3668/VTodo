# TodoAPP

## 简介

一个基于Vuetify + Vue的待办事项小工具

## 食用方法

### 安装依赖
```
yarn install
```

### 开发环境启动
```
yarn serve
```

### 生成环境构建
```
yarn build
```

### Lints and fixes files
```
yarn lint
```
